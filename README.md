# Stanford CS231n: Deep Learning for Computer Vision (Spring 2025)

This repository contains my solutions to the assignments for the [CS231n: Deep Learning for Computer Vision](http://cs231n.stanford.edu/) course at Stanford University (2025).




## Assignment 1: Neural Network Basics
Building the foundation of image classification using NumPy from scratch.
* **k-Nearest Neighbor (kNN):** Implementing L1 and L2 distances and cross-validation.
* **Support Vector Machine (SVM):** Vectorized loss functions and analytic gradients.
* **Softmax Classifier:** Cross-entropy loss and gradient descent optimization.
* **Two-Layer Network:** Building a modular MLP with a hidden layer and backpropagation.
* **Image Features:** Improving classification using HOG and Color Histograms.

## Assignment 2: Deep Learning Architecture
Moving toward modular design and deep convolutional networks.
* **Fully Connected Nets:** Modular layers for arbitrary depth.
* **Normalization:** Implementing Batch Normalization and Layer Normalization.
* **Regularization:** Implementing Dropout to improve generalization.
* **Convolutional Networks:** Manual implementation of Conv and Pool layers, followed by a deep "Sandwich" network.
* **PyTorch:** Introduction to deep learning frameworks on CIFAR-10.
* **RNN Captioning:** Image captioning using Vanilla Recurrent Neural Networks.

## Assignment 3: Generative Models & Transformers
Working with state-of-the-art architectures and generative AI.
* **Image Captioning with Transformers:** Using Multi-Head Attention to process visual features.
* **Self-Supervised Learning (SSL):** Learning representations without labels via SimCLR/MAE techniques.
* **Generative Models (Diffusion):** Implementing Denoising Diffusion Probabilistic Models (DDPM).
* **Foundation Models:** Exploring CLIP (Contrastive Language-Image Pre-training) and DINO.
